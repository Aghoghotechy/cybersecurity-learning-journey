## Key Concepts Learned

* System Configuration (MSConfig) can be used to troubleshoot startup and boot-related issues.
* Advanced System Settings provide access to performance, startup, recovery, and environment variable settings.
* User Account Control (UAC) helps reduce unauthorized system changes by requiring elevated permissions for certain actions.
* Computer Management provides centralized access to administrative tools and system utilities.
* System Information displays detailed information about hardware, software, and system components.
* Resource Monitor helps analyze CPU, memory, disk, and network activity.
* Command Prompt provides a command-line interface for system administration and troubleshooting.
* The Windows Registry is a hierarchical database that stores configuration information for users, applications, hardware, and the operating system.
