## Key Concepts Learned

* Effective searching is a critical cybersecurity skill.
* Shodan can identify internet-facing devices and exposed services.
* VirusTotal helps analyze suspicious files, URLs, and indicators of compromise.
* CVE databases provide information about publicly disclosed vulnerabilities.
* Technical documentation and MAN pages are valuable resources for understanding commands and tools.
* GitHub can contain Proof-of-Concept (PoC) code, vulnerability research, exploit demonstrations, and security tools.

## Practical Activities

* Investigated internet-connected systems using Shodan.
* Analyzed indicators using VirusTotal.
* Researched vulnerabilities through CVE databases.
* Used technical documentation to understand system commands.
* Examined a GitHub repository containing information about a simulated CVE.
