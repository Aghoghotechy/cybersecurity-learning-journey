## Key Concepts Learned

* Linux is one of the most widely used operating systems in cybersecurity and server environments.
* The terminal allows users to interact directly with the operating system using commands.
* Navigation through directories is performed using commands such as `pwd`, `cd`, and `ls`.
* Files and directories can be created, viewed, modified, and removed using command-line utilities.
* The `find` command can search for files and directories across the filesystem.
* Shell operators can combine commands and redirect output efficiently.
