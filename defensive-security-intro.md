## Topics Covered

- Security Operations Center (SOC)
- Monitoring and Detection

- Threat Investigation

## Key Concepts Learned

- Defensive security focuses on protecting systems and networks from attacks.
- Security teams monitor activity to detect suspicious behavior.
- SOC analysts investigate alerts and security events.
- Early detection helps reduce the impact of cyber attacks.
- Incident response involves identifying, investigating, containing, and resolving threats.

## Practical Activities

- Investigated suspicious network activity.
- Reviewed monitoring dashboard alerts.
- Identified a suspicious source IP address.
- Determined the type of attack being performed.
- Applied a defensive action to stop malicious activity.

## Reflection

This room introduced the responsibilities of defensive security professionals and SOC analysts. I learned how monitoring systems, alerts, and investigations are used to identify and respond to security threats before they cause significant damage.
